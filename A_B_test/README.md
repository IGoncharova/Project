# Проект по А/B-тестированию

### Задача: 
Провести оценку результатов A/B-теста.

### Техническое задание

- Название теста: recommender_system_test;
- Группы: А (контрольная), B (новая платёжная воронка);
- Дата запуска: 2020-12-07;
- Дата остановки набора новых пользователей: 2020-12-21;
- Дата остановки: 2021-01-04;
- Аудитория: 15% новых пользователей из региона EU;
- Назначение теста: тестирование изменений, связанных с внедрением улучшенной рекомендательной системы;
- Ожидаемое количество участников теста: 6000.
- Ожидаемый эффект: за 14 дней с момента регистрации в системе пользователи покажут улучшение каждой метрики не менее, чем на 10%:
  - конверсии в просмотр карточек товаров — событие product_page
  - просмотры корзины — product_cart
  - покупки — purchase.

## Общий вывод

В рамках работы провела исследовательский анализ данных, а так же оцентку корректности проведеного теста и проверку одной статистической гепотезы о различии между группами A и B по всем событиям воронки. Ожидаемый эффект, в новой платёжной группе не был достигнут.
При этом в ходе работы над проектом обнаружены множественные нарушения в проведении теста, некоторые условия тех.здания не были выполнены.  По итогу дала рекомендации не считать результат теста верными и запустить его повторно при корректном соблюдении условий.
